# Me: ตัวตนเจ้าของโปรเจกต์ (jj)

- **Workflow is King:** ให้ความสำคัญกับลำดับการทำงานของคนหน้างาน (Worker Flow) มากกว่าความสวยงามแบบศิลปะ
- **Simple & Direct:** ชอบความเรียบง่าย ไม่รกรุงรัง ไม่ชอบตัวเลือกเยอะหรือคำอธิบายที่ยาวเกินความจำเป็น
- **UI Preferences:** 
    - ชอบ Layout แบบ 3-Bar (ซ้าย-กลาง-ขวา)
    - ชอบตารางที่ใช้งานได้เหมือน Excel (Sort, Filter, Freeze)
    - ชอบโทนสีเขียวอ่อนที่ดูสบายตา
    - UI ต้องเป็นภาษาไทยสำหรับ Worker และภาษาอังกฤษสำหรับ Admin Monitor
- **No Nonsense:** ต้องการผู้ช่วยที่พูดตรง ทำงานไว ไม่ต้องมีพิธีรีตอง และไม่ข้าม Scope งานเอง
- **System Integrity:** ต้องการระบบที่มีการจัดการ Logs, Backup และ Data ที่เป็นระเบียบชัดเจน
- **User vs Admin:** แยกหน้าจอคนงาน (Worker) ออกจากส่วนตั้งค่า (Admin) อย่างเด็ดขาด
